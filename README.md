# Template Ignite

YAML/JSON 설정 기반 정적 랜딩 페이지 자동 생성 시스템

## 기술 스택

- **Next.js 14+** (App Router, SSG)
- **TypeScript** (엄격한 타입 시스템)
- **StyleX** (Zero-runtime CSS-in-JS)

## 프로젝트 구조

```
claudedocs/           # 프로젝트 설계 문서
├── PROJECT_DESIGN.md
├── PROTOTYPE_GUIDE.md
├── TYPE_DEFINITIONS.md
├── SAMPLE_TEMPLATES.md
└── NEXT_SESSION.md
```

## 시작하기

자세한 구현 가이드는 `claudedocs/PROTOTYPE_GUIDE.md` 참고

## 목적

1인 기업 및 소규모 비즈니스를 위한 빠른 랜딩 페이지 생성
