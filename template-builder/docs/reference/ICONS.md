# 아이콘 가이드 (Icons Guide)

Template Builder에서 사용 가능한 Lucide React 아이콘 전체 목록입니다.

## 사용 방법

JSON 템플릿에서 아이콘을 사용할 때는 아이콘 이름을 문자열로 지정합니다:

```json
{
  "icon": "CheckCircle",
  "title": "확인됨"
}
```

## 참고사항

1. **모든 아이콘은 PascalCase로 작성**
   - 올바름: `CheckCircle`
   - 잘못됨: `checkCircle`, `check-circle`

2. **아이콘 크기는 자동 조정**
   - 컴포넌트에서 자동으로 적절한 크기로 렌더링

3. **색상은 테마에 따라 자동 적용**
   - 별도 색상 지정 불필요

4. **더 많은 아이콘**
   - 전체 목록: https://lucide.dev/icons
   - 500+ 아이콘 사용 가능

## 예제 사용

### Features 섹션
```json
{
  "type": "features",
  "variant": "FeaturesThreeColumn",
  "content": {
    "title": "핵심 기능",
    "subtitle": "우리의 강점",
    "items": [
      {
        "icon": "Zap",
        "title": "빠른 속도",
        "description": "최적화된 성능"
      },
      {
        "icon": "Shield",
        "title": "안전한 보안",
        "description": "데이터 보호"
      },
      {
        "icon": "Users",
        "title": "팀 협업",
        "description": "효율적인 협업"
      }
    ]
  }
}
```

### Contact 섹션
```json
{
  "type": "contact",
  "variant": "ContactSplit",
  "content": {
    "contactInfo": [
      {
        "icon": "Phone",
        "label": "전화",
        "value": "02-1234-5678"
      },
      {
        "icon": "Mail",
        "label": "이메일",
        "value": "contact@example.com"
      },
      {
        "icon": "MapPin",
        "label": "주소",
        "value": "서울시 강남구"
      }
    ]
  }
}
```

## 아이콘 선택 팁

1. **일관성 유지**: 같은 카테고리 내에서는 유사한 스타일의 아이콘 사용
2. **의미 명확성**: 사용자가 직관적으로 이해할 수 있는 아이콘 선택
3. **과용 방지**: 너무 많은 아이콘은 오히려 혼란을 줄 수 있음
4. **업종 적합성**: 각 업종에 맞는 아이콘 사용
5. **테스트**: 실제 사용 환경에서 가독성 확인
